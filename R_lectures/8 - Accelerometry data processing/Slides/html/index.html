DATA TREATMENT - SLEEP
Usando el dataset de 60 sec epoch:
i. Se clasifican los minutos como sueño/despierto (Sadeh et al., 1994)
ii. Se identifican periodos de sueño continuos (7:00 pm – 6:00 am)
• Mínimo: 160 min
• Tolerancia: 10/20 min (continuos)
iii. Se buscan periodos de no uso clasificados como periodos de sueño.
• Se identifican usando los counts del eje vertical (axis 1)
• Mínimo: 90 min
• Tolerancia: 2 min (no continuos)
